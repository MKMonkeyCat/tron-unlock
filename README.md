# TronUnlock - TronClass 前端研究與功能優化腳本

<img src="./public/logo.png" alt="TronUnlock Logo" width="200" />

> 使用 TypeScript + Vite + Preact 開發的前端用戶腳本，針對 TronClass 系列學習平台進行前端行為分析與功能優化研究。  
> ⚠ 本專案僅供學術研究與學習用途，**禁止用於任何未授權或非法行為**。

![Script Settings](./.github/assets/script-setting.png)

## 支援平台

腳本會在 TronClass 官方網域（`tronclass.com` / `.com.tw` / `.com.cn` 及其子網域）以及部分學校自架站台上自動啟用，完整清單見 [const.ts](./const.ts) 的 `LEARNING_PLATFORM_DOMAINS`。若你所屬的平台網域不在清單內，可自行新增後重新建置。

## 環境需求

- Node.js（版本見 [.nvmrc](./.nvmrc)，建議使用 `nvm use` 對齊版本；若無 nvm，需 Node.js 20 以上）
- npm
- 依安裝方式擇一：
  - Tampermonkey 腳本：瀏覽器安裝 [Tampermonkey](https://www.tampermonkey.net/)
  - Chrome 擴充功能：Chromium 系瀏覽器（Chrome / Edge 等），支援 Manifest V3

## 安裝步驟

1. 下載或 clone 本專案：

```bash
git clone https://github.com/MKMonkeyCat/tron-unlock.git
cd tron-unlock
```

2. 安裝依賴套件：

```bash
npm install
```

3. 建置專案：

```bash
npm run build
```

建置會產生兩種輸出：

- `dist-userscript/main.user.js`：單一檔案，貼進 **Tampermonkey** 新增腳本即可（`npm run build:userscript` 可單獨產生）。
- `dist/`：Chrome 擴充功能（含 side panel 設定面板），到 `chrome://extensions` 開發者模式「載入未封裝項目」選擇這個目錄；`release/` 底下也會產生對應的 zip（`npm run build:extension` 可單獨產生）。

4. 依你使用的方式安裝上述其中一種（或兩種都裝）。

5. 開啟 **相關 ULearn/TronClass 網站**，腳本將自動啟用；可透過 Tampermonkey 選單或擴充功能的 side panel 開關個別功能。

## 開發

```bash
npm run dev            # Userscript 開發模式（Vite dev server）
npm run dev:extension  # 擴充功能開發模式，含自動重載
npm run lint            # ESLint 檢查
npm run lint:fix        # ESLint 自動修正
npm run test            # 執行 vitest 測試
npm run test:type       # 型別檢查
```

## 功能概述

所有功能以「頁籤（全域 / 課程 / 考試）→ 分類 → 功能」三層組織，可在擴充功能選單中逐項獨立開關；部分功能另有細部設定欄位。

### 全域 › 事件攔截

攔截頁面對瀏覽器事件的監聽與偵測，避免平台以這些事件判斷使用者是否切走視窗、開啟開發者工具等。

| 功能               | 說明                                                                                                                               |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------- |
| 分頁切換偵測繞過   | 攔截 `blur` / `focus` / `beforeunload` / `unload` / `pagehide` / `pageshow` 等事件，避免頁面偵測到視窗或分頁切換、失去焦點等行為。 |
| 開發者工具偵測繞過 | 停用頁面內建的開發者工具偵測機制，避免因開啟 DevTools 或除錯而觸發警示或限制。                                                     |
| 閒置偵測繞過       | 模擬人類滑鼠移動行為以維持「使用中」狀態，並隱藏平台彈出的閒置警告視窗。                                                           |
| 複製貼上限制繞過   | 解除頁面對複製、剪下、貼上與右鍵選單的限制，並還原 `Ctrl/Cmd + C/V/X` 快捷鍵。                                                     |
| 全螢幕偵測繞過     | 攔截全螢幕狀態變化事件與 `F11` 快捷鍵，避免頁面偵測到退出全螢幕。                                                                  |

### 全域 › 樣式

| 功能         | 說明                                                           |
| ------------ | -------------------------------------------------------------- |
| 課程頁面樣式 | 調整課程頁面版面配置，讓內容區塊更寬、字體更大，改善閱讀體驗。 |
| 考試頁面樣式 | 調整考試頁面版面配置，讓內容區塊更寬、字體更大。               |

### 全域 › 雜項

| 功能           | 說明                                                         |
| -------------- | ------------------------------------------------------------ |
| 隱藏頁腳       | 隱藏頁面底部的頁腳區塊並移除對應留白空間。                   |
| 載入時隱藏捲軸 | 頁面完全載入前先隱藏捲軸，避免版面因樣式尚未套用完成而閃動。 |

### 課程 › 學習活動

| 功能         | 說明                                                                                  |
| ------------ | ------------------------------------------------------------------------------------- |
| 強制允許下載 | 攔截 `GET /api/activities/:id` 回應，將 `allow_download` 欄位強制改為 `true`。        |
| 強制允許快轉 | 攔截同一 API 回應，將 `allow_forward_seeking` 欄位強制改為 `true`，解除影片快轉限制。 |
| 自動觀看影片 | 自動播放課程影片，並依設定門檻自動標記為已觀看；可調整以下欄位：                      |

自動觀看影片的細部設定：

| 欄位         | 說明                                                                 | 預設值 |
| ------------ | -------------------------------------------------------------------- | ------ |
| 自動開始播放 | 影片載入後自動開始播放，不需手動按下播放鍵。                         | 開啟   |
| 播放速率     | 影片預設播放速度（範圍 0.1 ~ 16，step 0.1）。                        | 1.75   |
| 門檻模式     | `auto` 由系統自動判斷完成門檻；`custom` 使用下方「觀看門檻」設定值。 | custom |
| 觀看門檻     | 播放進度達此比例（0 ~ 1）即視為已完成觀看。                          | 1      |
| 門檻隨機化   | 開啟後門檻會加入隨機浮動，避免每次都在同一秒數完成觀看判定。         | 開啟   |

### 考試 › 雜項

| 功能       | 說明                                                                                           |
| ---------- | ---------------------------------------------------------------------------------------------- |
| 隱藏浮水印 | 隱藏考試頁面上以 `#symbol-water-mark` 等元素呈現的浮水印（含 `::before` / `::after` 偽元素）。 |
